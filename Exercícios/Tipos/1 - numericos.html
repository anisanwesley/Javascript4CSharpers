Desafios Number

Duvido você Resolver esses Numeros

1. Par

Crie uma function que retorne se o numero é par:

  • Input: 2 output: true
  • Input: 5 output: false

2. Fatorial

Crie uma function que retorne o fatorial de um número:

  • Input: 4 output: 24
  • Input: 8 output: 40320

3. Somar tudo

Crie uma function que some números desde o 1 até o número passado por argumento:

  • Input: 12 output: 78
  • Input: 140 output: 9870

4. Verificar Números

Crie uma function que receba dois números, retorne true se o primeiro for maior que o segundo. Caso sejam iguais, retorne null:

  • Input: 12 output: 78
  • Input: 140 output: 9870
