Fácil

1. Sopa de letras

Crie uma function que retorne uma palavra com as letras ordenadas por ordem alfabética, ignorando espaços e caracteres especiais:

  • Input: 'javascript' output: 'aacijprstv'
  • Input: 'a casa de cera' output: 'aaaccdeers’'

2. Contar vogais

Crie uma function que retorne o número de vogais em uma sentença:

  • Input: 'javascript' output: 3'
  • Input: 'a casa de cera' output: 6

3. Potência de 2

Crie uma function que retorne se o número é potência de 2:

  • Input: 32 output: true
  • Input: 18 output: false

4. Primo

Crie uma function que diga se o número é primo:

  • Input: 8 output: true
  • Input: 23 output: false

Médio

5. Palindromas

Crie uma function que verifique se a sentença a é palíndroma:

  • Input: a torre da derrota output: true
  • Input: arara output: true

6. Inverter

Crie uma function que retorne uma string invertida:

  • Input: "eu gosto de ver minha cachorra pular" output: "ralup arrohcac ahnim rev ed otsog ue"
  • Input: "all you need is code" output: "edoc si deen uoy lla"

7. MDC

Crie uma function que retorne o maior divisor comum entre dois numeros:

  • Input: 7, 3 output: 1
  • Input: 36, 54 output: 18

Difícil

8. Recursividade

Crie uma function que retorne a soma de todos os itens numéricos de um array, se este item for um array, deverá somar seus itens internos também também :

  • Input: [1,2,3,4,5] output: 15
  • Input: [1,[2,3],4,5] output: 15

9. Trocar Letras

Crie uma function que substitua cada caractere da string pela próxima letra no alfabeto:

  • Input: "ABcde" output: "BCdef"
  • Input: "a casa de cera" output: "b!dbtb!ef!dfsb"

Dica: Use String.fromCharCode(n) e “texto”.charCodeAt() para realizar as conversões

10. Converter Binário

Crie uma function que converta um numero binário em decimal:

  • Input: 100101 output: 37
  • Input: 011 output: 3
