Desafios Arrays

Duvido você Resolver esses Arrays

1. As pontas

Crie uma function que retorne o maior e o menor número de um array em outro array:

  • Input: [4,1,2,6,5,8,7,4] output: [1,8]

2. Repetir

Crie uma function que retorne a quantidade números repetidos em um array:

  • Input: [1,2,3,2,4,5] output: 1
  • Input: [1,1,2,2,3] output: 2

3. Penúltimo

Crie uma function que retorne o segundo maior valor:

  • Input: [1,4,7,6,3,9] output: 7

4. Maior palavra

Crie uma function que retorne o a maior palavra da frase, ou um array com elas caso der empate:

  • Input: "eu gosto de ver minha cachorra pular" output: 'cachorra'
  • Input: "a casa de cera" output: ['casa','cera']

5. Alunos ativos

Crie uma function que retorne a quantidade de alunos ativos obtida de um array de alunos:

  • Input: [{ativo:true},{ ativo:true },{ ativo:false }] output: 2
