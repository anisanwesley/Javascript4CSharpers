Desafios Strings

Duvido você Resolver essas Strings

1. Capitalizar

Crie uma function que deixe a primeira letra de cada palavra em maiúsculo:

  • Input: ‘all you need is code’ output: ‘All You Need Is Code’

2. Converter tempo

Crie uma function que receba a quantidade de segundos e retorne o valor em minutos, formatado :

  • Input: 50 output: '0:50'
  • Input: 140 output: '2:20'

3. Contar palavras

Crie uma function que retorne a quantidade de palavras de uma sentença:

  • Input: 'javascript' output: 1
  • Input: 'a casa de cera' output: 4

4. Calcular Segundos

Crie uma function que retorne a quantidade de segundos de um determinado horário:

  • Input: "00:00:57" output: 57
  • Input: "01:01:01" output: 3661

5. Formatar Telefone

Crie uma function que retorne uma máscara de telefone internacional para uma serie de numeros:

  • Input: 554912345678 output: "+55 (49) 1234-5678"

6. Validar extensão de arquivo

Crie uma function que retorne a extensão do arquivo passado via string, ou false caso inválido:

  • Input: 'teste.jpeg' output: 'jpeg'
  • Input: 'unnamed' output: false
